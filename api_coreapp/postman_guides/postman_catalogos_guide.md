# Guía de Postman - Endpoints de Catálogos

Usa estos ejemplos para obtener los valores de los catálogos en Postman. Son muy útiles para poblar dropdowns o selects en el frontend.

> [!IMPORTANT]
> Estos endpoints están protegidos. Necesitas enviar el **Bearer Token** en la pestaña *Authorization*.

---

## 1. Tipos de Torneo (`GET /api/catalogos/tipos-torneo`)
**URL:** `{{base_url}}/catalogos/tipos-torneo`
*(No requiere body)*

## 2. Categorías (`GET /api/catalogos/categorias`)
**URL:** `{{base_url}}/catalogos/categorias`
*(No requiere body)*

## 3. Estados de Partido (`GET /api/catalogos/estados-partido`)
**URL:** `{{base_url}}/catalogos/estados-partido`
*(No requiere body)*

## 4. Tipos de Multa (`GET /api/catalogos/tipos-multa`)
**URL:** `{{base_url}}/catalogos/tipos-multa`
*(No requiere body)*
